# QuoteCraft

Internal quoting and CRM tool for roofing & exterior restoration contractors.

## Stack

- **React 19** + **Vite**
- **Tailwind CSS v4**
- Planned: Next.js, Supabase, Prisma, Tiptap

## Getting Started

```bash
npm install
npm run dev
```

## Features (Prototype)

- Dashboard with KPI cards, recent quotes, follow-up queue, trade volume chart
- Quote list with search and status filtering
- New Quote wireframe (full creation form)
- Existing Quote editor wireframe (line items, tabs, sidebar summary, activity log)
- Collapsible sidebar navigation
